# Моё портфолио 🚀

Современный сайт-портфолио веб-разработчика с тёмной темой и анимациями.

## ✨ Особенности

- 🎨 Тёмный градиентный дизайн
- 📱 Полностью адаптивная вёрстка
- ⚡ Плавная прокрутка к секциям
- 🎯 Карточки навыков с иконками
- 📝 Подробные комментарии к коду

## 🌐 Демо

[Посмотреть сайт](https://k9anak.github.io/test/portfolio.html)

| Файл | Описание |
|------|----------|
| `portfolio.html` | Сайт-портфолио |
| `index.html` | Страница с приветствием |
| `README.md` | Документация |

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, градиенты, анимации)
- JavaScript (плавная прокрутка)

## 🚀 Развёртывание

### Через GitHub Pages:
1. Зайдите в **Settings** репозитория
2. Перейдите в раздел **Pages**
3. Включите GitHub Pages для ветки `main`

### Локально:
Просто откройте `portfolio.html` в браузере.
